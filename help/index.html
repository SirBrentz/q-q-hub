Help Center

Documentation and support for the Quidget WooCommerce plugin.

Setup & Installation

System requirements

WordPress 5.0+, WooCommerce 4.0+, PHP 7.2+, SSL certificate (HTTPS), a publicly accessible site, and an active Qualiphy account with API key.

Installing the plugin

In your WordPress admin, go to Plugins → Add New → Upload Plugin. Upload the qualiphy.zip file, click Install Now, then Activate. The setup wizard launches automatically.

Setup wizard

The wizard walks you through: accepting terms, verifying requirements (SSL, WooCommerce), uploading your branding, connecting your API key, selecting your service area, importing consultations as WooCommerce products, and configuring settings.

Finding your API key

Log in at app.qualiphy.me, click Settings in the left sidebar. Your API key is at the top under "API Key (Default Clinic)" — click Copy. No account yet? Sign up free.

Do I need WooCommerce Subscriptions?

Only for recurring monthly consultations (like weight loss refills). One-time consultations and GFE exams work without it.

Payment gateway

Required for paid consultations. Stripe, PayPal, or similar must be active in WooCommerce → Settings → Payments. Free consultations work without a gateway.

Settings & Configuration

Auto Send & Auto Launch

Auto Send (default: enabled) sends exam invites automatically on checkout. Auto Launch (default: enabled) shows the exam modal or redirect on the thank-you page. Disable either to use manual triggers instead.

Popup vs Redirect

Popup shows a modal with meeting link, QR code, and dismiss option. Redirect counts down then sends the patient to the meeting URL. Configure under Qualiphy → Settings → General.

Auto Refund & Return Fee

Deferred consultations auto-refund the order total minus a return fee (default $30). The fee is overridable per product. Per-item fees are summed across Qualiphy line items at checkout.

Default Price Margin

Auto-fills patient-facing price during import. Default 10% — if Qualiphy cost is $28, patient price fills to $30.80. Adjust before importing.

Checkout fields

Gender, Date of Birth, and Phone are added automatically when a Qualiphy product is in the cart. Guest checkout is disabled — accounts are created automatically. Age restrictions: Rx requires 18+, GFE requires 13+.

Service area

48 states + D.C. (Alabama and Mississippi excluded). Configure under Qualiphy → Settings → Service Area. A WooCommerce shipping zone is created automatically.

Branding

Upload light and dark logo variants and set accent colors under Qualiphy → Settings → Branding. These appear on the exam launch modal and patient emails. PNG with transparent background recommended.

Language & text customization

All patient-facing text is editable: button labels, modal copy, questionnaire text, dose labels, and disclaimers. Found under Qualiphy → Settings → Language.

Clinical Access controls

Only users with clinical access can view patient order details. The setup admin gets access automatically. Non-clinical users see "Patient Order Access Required." Manage in Settings.

Email notifications

Admin emails for invite failures include error details and retry instructions. Patient emails are sent automatically. Configure under Qualiphy → Settings → Emails.

Troubleshooting

Exam invite never sends after checkout

No payment gatewayInvites fire on payment confirmation. Check WooCommerce → Settings → Payments.
CachingExclude /checkout/ and /my-account/ from your cache plugin.
Free exam misconfiguredFree consultations must use "Free Instant Checkout" during import.

"Please log in to view this order"

Account creation disabledWooCommerce → Settings → Accounts → enable account creation during checkout.
CacheClear your entire site cache after installing the plugin.

Exam popup doesn't appear

Auto-launch disabledEnable Auto Launch Exams in Qualiphy → Settings.
Missing meeting URLCheck the order's Qualiphy section for error details.

"Not available in your area"

Service areaCheck states under Qualiphy → Settings → Service Area.
Shipping zone deletedRe-save service area to regenerate the WooCommerce zone.

Prescription not created after approval

Pharmacy routingVerify pharmacy assigned for the patient's state under Qualiphy → Pharmacies.
Blocked webhookCheck if a security plugin is blocking incoming POST requests.

WP 6.7+ translation notice

Known issue, fixed in v1.1.0. Update the plugin. The notice is cosmetic and doesn't affect functionality.

FAQ

Why can't patients skip checkout?

Checkout collects required patient data (name, gender, DOB, phone, state). The plugin auto-redirects and auto-creates accounts. For staff use, create a direct link: yoursite.com/checkout/?add-to-cart=PRODUCT_ID

Can I rename products after importing?

Yes. The Qualiphy link is stored as metadata, not tied to the product title. Rename freely.

Which states does Qualiphy cover?

48 states + D.C. Alabama and Mississippi are currently excluded.

Can I have multiple consultations in one cart?

One Rx consultation per cart (for proper exam delivery). Multiple GFE exams are allowed in a single session.

How do I reset the plugin?

Go to Plugins → Uninstall Qualiphy → reactivate. This returns you to the setup wizard. Your Qualiphy account data is unaffected.

Sandbox vs Live mode

Sandbox simulates the system without real charges — use it during setup and testing. Live processes actual patient exams. Switch to Live after running a successful test order.
