Getting Started

Everything you need to know to install, activate, and configure the Quidget plugin on your WordPress site.

Installing the Plugin

Quidget integrates seamlessly with WordPress and WooCommerce. Before installing, make sure your site meets the following requirements:

Requirements

  • WordPress 6.0+ (we recommend the latest version)
  • WooCommerce 8.0+ (required for product and order management)
  • PHP 7.4+ (PHP 8.0+ recommended for better performance)

Installation Method 1: Upload via WordPress Admin

  1. Log in to your WordPress admin dashboard
  2. Navigate to Plugins → Add New
  3. Click the Upload Plugin button at the top
  4. Select the Quidget plugin ZIP file from your computer
  5. Click Install Now and wait for the installation to complete
  6. Click Activate Plugin to enable Quidget on your site

Installation Method 2: WordPress Plugin Directory

  1. In your WordPress admin, go to Plugins → Add New
  2. Search for "Quidget" in the search box
  3. Click Install Now on the Quidget plugin result
  4. Once installed, click Activate Plugin

After activation, you'll see the Quidget menu appear in your WordPress admin sidebar. You're ready to move on to the next steps!

Activating Your License

Quidget is a free plugin to install and activate — no license key required. Billing is handled per-consultation through your Qualiphy account, so you only pay when patients complete consultations.

How Licensing Works

The plugin automatically activates upon installation. There's no separate license activation step or license key to enter. Instead, Quidget connects to your Qualiphy account via API key (see the next section), and consultations are billed directly to your Qualiphy account on a per-use basis.

Billing Information

Consultation charges appear in your Qualiphy account under Settings → Billing → Invoice History. Each consultation is itemized with the consultation type, patient information, and cost. You can download invoices and set up automated payments through your Qualiphy account dashboard.

Next: Connect your Qualiphy API key to start linking consultations to your clinic.

Connecting Your Qualiphy API Key

The Quidget plugin communicates with your Qualiphy clinic account through an API key. This connection ensures that exams are routed correctly and consultation data syncs automatically.

Step 1: Get Your API Key from Qualiphy

  1. Log in to your Qualiphy clinic account at app.qualiphy.me
  2. Go to Settings → API
  3. Copy your API key (it's a long alphanumeric string)
  4. Keep this key safe — treat it like a password

Step 2: Enter the API Key in Quidget

  1. In your WordPress admin, go to Qualiphy → Settings → General
  2. Paste your Qualiphy API key into the API Key field
  3. Click Verify & Save

Verification Success

If the connection is successful, you'll see a green checkmark next to your API key field and a confirmation message. Your plugin is now connected to your Qualiphy clinic.

Don't Have a Qualiphy Account?

If you don't yet have a Qualiphy account, sign up for free at app.qualiphy.me/signup. You'll create your clinic profile and be given an API key immediately.

Next: Run the setup wizard to configure your branding and service area.

Running the Setup Wizard

The Quidget setup wizard launches automatically after you successfully connect your Qualiphy API key. It guides you through the essential configuration steps to get your store ready for consultations.

Step 1: Branding

Upload your clinic logo and choose accent colors. These will appear on the exam modal, checkout pages, and patient emails:

  • Clinic Logo: PNG with transparent background recommended (light and dark variants)
  • Primary Color: Default is #CFC9FF (light purple) — used for buttons and accents
  • Secondary Color: Default is #6e54a7 (dark purple) — used for text and highlights

Step 2: Service Area

Select the states where you're licensed to provide consultations. Quidget supports 48 states plus Washington, D.C. (Alabama and Mississippi are not currently available). The plugin automatically creates a WooCommerce shipping zone for your service area, and customers outside your area will see "not available in your area."

Step 3: Import Consultations

Choose to either create new products for your consultations or link to existing WooCommerce products. The wizard will help you set pricing, exam type (GFE or Rx), and pharmacy routing if applicable.

Step 4: Review

Review all your settings before finalizing. You can edit anything after completing the wizard from the Qualiphy settings pages.

Re-running the Wizard

Need to re-run the setup wizard? Go to Qualiphy → Settings → General and click Run Setup Wizard at the bottom of the page. This won't overwrite existing settings — you can choose to update them selectively.

Next: Create your first test consultation to verify everything is working.

Your First Consultation

Now that Quidget is installed and configured, it's time to test the full flow end-to-end. Creating a test consultation ensures that exam invites are sent correctly and the order lifecycle works as expected.

Step 1: Create a Test Order

  1. Go to your Quidget storefront (the public WordPress site)
  2. Add a consultation product to your cart
  3. Proceed to checkout
  4. Fill in patient information: name, email, DOB, and any other required fields
  5. Complete payment (use Stripe test cards if in test mode)

Step 2: Check the Exam Invite

Within seconds of payment, an exam invite email will be sent to the email address entered at checkout. Open your email (check spam/promotions if needed) and verify that:

  • The invite contains your clinic logo and branding colors
  • The invite includes a link to start the exam
  • The email is from your clinic's email address

Step 3: Complete the Exam Flow

  1. Click the exam invite link
  2. Complete the exam questionnaire (demographic info, medical history, etc.)
  3. You'll see a "Exam Submitted" confirmation and a meeting link for the provider consultation

Step 4: Check the Order in WooCommerce

  1. Go to WooCommerce → Orders in your WordPress admin
  2. Find the test order you just created
  3. Check the Qualiphy section on the order detail page
  4. Verify that the exam status shows the current state (e.g., "Submitted", "Pending Approval")

Troubleshooting

  • Exam invite not received: Check spam folders, or verify the email address at checkout
  • Order not showing Qualiphy data: Refresh the page, or check that your API key is still connected
  • Exam modal not opening: Clear browser cache, or check that JavaScript is enabled

Congratulations! You've successfully set up Quidget and completed a test consultation. You're ready to launch to patients. For help with settings, see the Plugin Settings guide.
