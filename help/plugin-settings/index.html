Plugin Settings

A complete reference for all Quidget plugin settings. Access settings from Qualiphy → Settings in your WordPress admin.

General Checkout Branding Questionnaire Service Area Language Clinical Access Email Redirects Products

General Settings

Access general settings from Qualiphy → Settings → General. These core settings control plugin behavior and API connectivity.

Enable Qualiphy

Default: Enabled (checked)

Master toggle for the Quidget plugin. When disabled, all Qualiphy consultation products are hidden from customers and the plugin stops processing exams and orders. Use this to temporarily disable the plugin without deactivating it.

Qualiphy API Key

Your encrypted Qualiphy API key for clinic authentication. This key is encrypted using AES-256-CBC encryption and is required for all API calls to sync consultations, send exam invites, and process webhooks. See Connecting Your Qualiphy API Key for setup instructions.

Advanced Settings

Default: Disabled (unchecked)

Enable to access advanced configuration options for developers and experienced users (webhook URL configuration, debug logging, custom field mappings, etc.).

Disable SSL Monitoring

Default: Disabled (unchecked)

Disables SSL certificate verification when making API requests to Qualiphy. Only enable this in development environments or if you encounter SSL certificate errors. Never enable in production.

Hide Custom Pharmacy Consultations

Default: Disabled (unchecked)

When enabled, consultations from custom pharmacy partners (outside the standard Qualiphy network) are not synced or displayed. Use this if you want to limit consultations to Qualiphy's verified pharmacy network.

Checkout Settings

Access checkout settings from Qualiphy → Settings → Checkout. These control the customer checkout experience and patient data collection.

Auto Launch Exams

Default: Enabled

When enabled, the exam modal automatically opens on the order thank-you page after payment is completed. Patients can immediately begin their exam without needing to click a link. Disable to show a simpler thank-you page with a manual "Start Exam" button.

Exam Launch Mode

Options: Popup or Redirect

  • Popup (Modal): Opens the exam in a modal dialog overlaid on the thank-you page. Patients see the exam details and can close the modal to return to your site.
  • Redirect: Redirects the patient to a dedicated exam page with a countdown timer until the video consultation begins. More immersive experience.

Recommended: Popup mode for most sites—it keeps patients engaged with your brand while showing exam details.

Force Redirect on Popup

Default: Disabled | Range: 0-10 seconds

When enabled with a time value, the exam modal automatically redirects to the full exam page after the specified seconds. Only applies when Exam Launch Mode is set to Popup. Use 0 to disable auto-redirect.

Checkout Redirect Delay

Default: 3 seconds | Range: 1-10 seconds

Delay before redirecting to the exam page when Exam Launch Mode is Redirect. Allows time for the thank-you message to display. Higher values give customers more time to see the order confirmation.

Auto Send Exams

Default: Enabled

When enabled, exam invites are automatically sent to patients immediately after payment is confirmed. Disable if you want to manually review orders before sending exams (useful for fraud prevention or high-value consultations).

Auto Refund Rejected

Default: Enabled

When enabled, if a provider rejects an exam, the patient's payment is automatically refunded to their original payment method. Disable to manually process refunds on rejected exams.

Exam Price Retained

Default: $30.00

The exam fee that Quidget retains when a patient is approved. This amount is deducted from the total consultation fee and kept by your clinic. The remainder is routed to Qualiphy and the pharmacy partner. You can override this per-product during import wizard or product settings.

Show Exam Refund Message

Default: Enabled

When enabled, checkout displays a message explaining the refund policy if an exam is rejected. Helps set patient expectations about potential refunds.

Branding & White-Label

Customize all patient-facing elements to match your clinic's branding. Access from Qualiphy → Settings → Branding.

Light Background Logo

Your clinic logo for use on light backgrounds (exam modal, checkout). Upload as PNG with transparent background. Recommended dimensions: 200x200px or larger.

Dark Background Logo

Dark version of your logo for use on dark backgrounds (patient emails, thank-you page). Upload as PNG with transparent background.

Primary Accent Color

Default: #CFC9FF (light purple)

Used for buttons, links, highlights, and the exam modal header. Choose a color that complements your brand and has good contrast on white backgrounds. Click the color picker to select.

Secondary Accent Color

Default: #6e54a7 (dark purple)

Used for text, headings, and secondary UI elements. Should have strong contrast with light backgrounds. Choose a color that pairs well with your primary color.

Applied Locations

Your branding is applied to:

  • Exam modal header, buttons, and accents
  • Checkout page (section headers, accent lines, buttons)
  • Patient thank-you page
  • Exam invite emails
  • Prescription/fulfillment notification emails

Color Picker Tips

  • Test colors on both light and dark backgrounds to ensure readability
  • Use a contrast checker tool (like WebAIM) to verify WCAG AA compliance for accessibility
  • Consider your brand guidelines when selecting colors
  • Light colors work better as primary (for backgrounds), dark colors as secondary (for text)

Questionnaire Style

Customize the appearance of the dose questionnaire that patients fill out. Access from Qualiphy → Settings → Questionnaire.

Understanding the Dose Questionnaire

The dose questionnaire is a 3-step guided form for returning patients to select their preferred dose/strength and adjustment preference. It appears during checkout for consultation types that support variable dosing (e.g., GLP-1 medications, HRT).

Bottle Colors Configuration

The questionnaire uses colored bottle SVGs to represent dose adjustment preferences. You can customize each bottle's appearance:

Stay (No Change)

  • Fill Color: Background color of the bottle icon (default #ede9f3 — light purple)
  • Stroke Color: Outline of the bottle (default #7c6aa0 — medium purple)

Increase Dose

  • Fill Color: Background color (default #e8f5e9 — light green)
  • Stroke Color: Outline (default #66bb6a — medium green)

Decrease Dose

  • Fill Color: Background color (default #fff3e0 — light orange)
  • Stroke Color: Outline (default #ffa726 — medium orange)

Live Preview

As you adjust colors, the questionnaire preview updates in real-time, showing how the bottles will appear to patients. This lets you verify your color selections before saving.

Best Practices

  • Use colors that are clearly distinct from each other (Stay, Increase, Decrease should be visually different)
  • Ensure sufficient contrast between fill and stroke colors
  • Use colors that align with common associations (green for increase, orange for decrease, purple for neutral)
  • Test on different devices and screen sizes

Service Area Configuration

Define the geographic region where you can legally provide consultations. Access from Qualiphy → Settings → Service Area.

Available States

You can select from 48 US states plus Washington, D.C. (50 jurisdictions total). Alabama and Mississippi are not currently supported by the Qualiphy network.

Selecting Your Service Area

  1. Go to Qualiphy → Settings → Service Area
  2. Check the boxes for all states where you're licensed to provide consultations
  3. Click Save Service Area

Automatic Shipping Zone Creation

When you save your service area, Quidget automatically:

  • Creates a WooCommerce shipping zone called "Qualiphy Service Area" covering all selected states
  • Adds free shipping to this zone (consultations don't need physical shipping)
  • Creates a "Qualiphy Consultation" shipping class for all consultation products

Protecting Your Shipping Zone

The shipping zone is protected and will auto-recreate if deleted. This ensures customers in your service area always see accurate shipping (free shipping, available to proceed to checkout).

Outside Service Area

Customers attempting to purchase from outside your service area will see:

"This product is not available in your area"

This message prevents orders from customers you're not licensed to serve.

Updating Service Area

If you add new licenses or lose licensure in a state, update your service area selection and save. The shipping zone updates automatically.

Language & Custom Text

Customize all patient-facing text throughout the consultation flow. Access from Qualiphy → Settings → Language.

Product Description Text

Customize how consultation types are described to customers:

  • Rx Product Description: Text shown on prescription consultations (e.g., "Get a prescription from a provider")
  • Non-Rx (GFE) Product Description: Text shown on aesthetic/clearance consultations (e.g., "Get provider approval")

Checkout Messages

  • No Shipping Available Message: Message shown when a customer is outside your service area
  • Refund Disclaimer: Explanation of when and how refunds are processed if an exam is rejected

Button Labels

  • Begin Consultation: Label on the main CTA button to start an exam (default "Begin Consultation")
  • Get Started Now: Alternative or secondary button label

Exam Launch Modal Text

Customize the modal that displays when an exam is ready to launch:

  • Modal Heading: Title of the exam modal (e.g., "Your consultation is ready")
  • Modal Description: Instructions explaining what to do next
  • Launch Button Text: Text on the main launch button
  • Audio/Video Note: Disclaimer about camera and microphone requirements
  • QR Code Heading: Label for the QR code section (mobile access)
  • QR Code Description: Instructions for scanning the code
  • Email Note: Message about the exam invite being sent to email
  • Dismiss Text: Label on the close/dismiss button

Questionnaire Text

Customize the dose questionnaire flow:

  • Step 1 Title: "Have you taken this medication before?" prompt
  • Step 2 Title: "What is your current dose?" prompt
  • Step 3 Title: "Dose adjustment preference" prompt

Dose Option Labels

  • Stay Dose Label: Label for "no change" option (e.g., "Stay at current dose")
  • Increase Dose Label: Label for "increase" option (e.g., "Increase dose")
  • Decrease Dose Label: Label for "decrease" option (e.g., "Decrease dose")
  • Questionnaire Disclaimer: Legal/informational disclaimer shown in the questionnaire

Patient Request Consultation

Messages shown during the consultation request/booking flow:

  • Request Consultation Message: Instructions for patients to request a new consultation
  • Consultation Request Confirmation: Confirmation message after request is submitted

Translation & Localization

All text fields support full HTML formatting (bold, links, etc.) for clarity. Consider your audience when customizing language—clear, jargon-free explanations work best for patient education.

Clinical Access Controls

Restrict access to patient consultation data to authorized clinical users. Access from Qualiphy → Settings → Clinical Access.

Why Clinical Access Control Matters

Patient consultation data is sensitive medical information. Clinical access controls ensure only authorized providers and staff can view patient exams, medical history, and order details.

Eligible Roles

Only users with these WordPress roles can access patient consultation data:

  • Administrator: Full access (typically site owner/manager)
  • Shop Manager: Full access (WooCommerce role for store operators)
  • Editor: Full access (content management role)
  • Author: Full access (contributor role)

All other roles (Subscriber, Customer, Contributor, custom roles) are restricted and cannot view patient consultation data.

Initial Access Setup

During the setup wizard, the admin user who runs the wizard automatically receives full clinical access. No additional configuration is needed for this user.

Non-Clinical Users

Users without clinical access see the message:

"Patient Order Access Required. You don't have permission to view patient consultation data."

Hiding Qualiphy Orders

Non-clinical users can toggle "Hide Qualiphy Orders" in the WooCommerce Orders list to filter out consultation orders from their view. This declutters their order management if they handle other product types.

Order Notes & History

Clinical users see exam status updates in the order notes and timeline (e.g., "Exam submitted", "Approved by Dr. Smith", "Prescription created", "Shipped"). Non-clinical users don't see these notes.

Auditing & Compliance

All access to patient data is logged in the order activity timeline. You can review who accessed patient orders and when by viewing the order detail page.

Email Notifications

Configure which admin staff members receive email notifications about exam status and errors. Access from Qualiphy → Settings → Email Notifications.

Notification Recipients

Add email addresses of team members who should receive admin notifications. You can add multiple recipients—each will receive the same notification emails.

Notifications Sent

Admin email notifications are sent for:

  • Exam Invite Failures: When an exam invite email fails to send to a patient (connection error, invalid email, etc.)
  • Exam Status Updates: When a provider approves, defers, or rejects an exam
  • Webhook Errors: When Qualiphy tries to send an update but your site doesn't acknowledge it
  • API Connection Issues: When the plugin can't reach the Qualiphy API (may indicate downtime or firewall issues)

Adding Recipients

  1. Go to Qualiphy → Settings → Email Notifications
  2. In the recipients section, enter a name and email address
  3. Click Add Recipient
  4. Click Save Settings

Auto-Populate During Setup

During the setup wizard, the admin user's email is automatically added as a notification recipient. You can add or remove recipients at any time.

Email Frequency

Notifications are sent in real-time as events occur (not batched or delayed). This ensures you're aware of issues immediately.

Unsubscribe

To remove a recipient from notifications, delete their entry from the recipients list and save settings.

Redirect Settings

Configure where patients are redirected after an exam is completed (approved, rejected, etc.). Access from Qualiphy → Settings → Redirects.

Overview

Four redirect pages can be customized based on exam outcomes. Each can point to an existing page, a new page created by the wizard, or a custom URL.

Approved Exam Redirect

Where the patient goes after the provider approves their exam:

  • For Rx consultations: Typically a "Prescription Submitted" page explaining next steps (pharmacy will contact them)
  • For GFE consultations: Typically a "You're Approved" page with clearance confirmation

Rejected Exam Redirect

Where the patient goes if the provider rejects or defers their exam. Usually a page explaining the rejection reason and options (resubmit, contact support, request refund).

Not Available Redirect

Where the patient goes if they're outside your service area. Usually a page saying consultations aren't available in their location with information about service area expansion.

Missed Exam Redirect

Where the patient goes if they don't show up for their scheduled video consultation. Usually a page offering to reschedule or try again.

Setting Up Redirects

  1. Go to Qualiphy → Settings → Redirects
  2. For each redirect type, select a page or enter a custom URL
  3. You can use internal pages (select from dropdown) or external URLs (enter full URL including https://)
  4. Click Save Redirects

Default Pages

During setup wizard, default pages are created for each redirect. You can edit these pages to customize messaging or redirect to different pages later.

Custom Redirects

To redirect to an external site or a page on another domain, enter the full URL (e.g., https://example.com/approved).

Product Settings

Configure defaults for all Quidget consultation products. Access from Qualiphy → Settings → Products. Individual products can override these defaults.

Default Price Margin

Default: 10% (Range: 0-500%)

Default profit margin applied to Qualiphy consultation costs when creating products. If a Qualiphy consultation costs $50 and margin is 10%, the customer price is $55. Can be overridden per-product during import or product editing.

Hide View Cart Link

Default: Enabled

When enabled, the "View Cart" link is hidden after adding a consultation product to cart. Streamlines the purchase experience by pushing customers directly to checkout.

Show Branding

Default: Enabled

When enabled, clinic logo and accent colors are displayed on consultation product images, thank-you pages, and checkout. Disable to show minimal branding.

Enable Questionnaire by Default

Default: Enabled

When enabled, the dose questionnaire is shown to all customers (if the consultation type supports it). Disable to hide the questionnaire by default (can still enable per-product).

Questionnaire Redirect to Checkout

Default: Enabled

When enabled, after patients complete the dose questionnaire, they're immediately redirected to checkout. Disable to show a review/summary page before checkout.

Auto Redirect to Checkout

Default: Enabled

When enabled, adding a consultation product to cart automatically redirects to checkout (skipping the cart page). Disable if you want customers to review their cart or add multiple products.
